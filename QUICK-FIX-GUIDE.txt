🔴 WHY NOTIFICATIONS & COACHING INSIGHTS AREN'T WORKING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM:
────────
1. ❌ Coaching Insights (Phase 7) not triggering when coaches complete tasks
2. ❌ Regular notifications might also be empty (needs testing)

ROOT CAUSE:
───────────
Missing 2 environment variables on Railway backend:

  Missing: GROQ_API_KEY (for AI coaching analysis)
  Missing: COACHING_INSIGHTS_ENABLED=true (feature flag)

Without these, the code checks for them and returns early WITHOUT doing anything.


═══════════════════════════════════════════════════════════════════════════════
✅ HOW TO FIX IT (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Get Free Groq API Key
─────────────────────────────
1. Go to https://console.groq.com
2. Sign up (email only, NO credit card needed)
3. Find "API Keys" section
4. Copy your API key (format: gsk_xxx...)

STEP 2: Add to Railway
──────────────────────
1. Go to https://railway.app
2. Click "Coach Task Tracker" project
3. Click "Backend" service
4. Click "Variables" tab
5. Add these two:

   GROQ_API_KEY=gsk_YOUR_KEY_HERE
   COACHING_INSIGHTS_ENABLED=true

6. Click Deploy
7. Wait 30 seconds

STEP 3: Test It Works
──────────────────────
1. Login to https://coachtracker-theta.vercel.app
2. Create a task
3. Complete it as coach
4. Check notification bell (top right)
5. Should see "coaching_insights" notification with AI feedback
